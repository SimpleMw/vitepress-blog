解释：

java核心思想为面向对象，在内存中为一个一个的对象，序列化与反序列化为操作对象的一种方式

<font color=red>序列化：将 java对象 转换为 二进制序列</font>

<font color=red>反序列化：将 二进制序列 还原为 java对象</font>

序列化用途：

- 对象存储硬盘

- 对象网络传输
