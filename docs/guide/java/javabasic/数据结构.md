## 数组

### 声明方式

```java
//数据类型 []  数组名称 = new 数据类型[数组长度];
int[] ids = new int[20]
```

```java
//数据类型 [] 数组名称 = {数组元素1，数组元素2，......}
int[] ids = {1,2}
```

注：

1.数组是不可变的，创建的时候即确定了大小

2.数组元素下标是从0开始的

3.length属性记录数组的大小

4.特点(有序)

	- 插入快，以及根据下标查找快
	- 删除慢，当删除一个后后续的元素会向前移动
